### **Augury**

**Descripción:** Augury es una extensión para Chrome que permite inspeccionar la estructura de componentes de Angular, el estado del router y las dependencias de inyección.

**Descarga:** Augury en Chrome Web Store

### **Lighthouse**

**Descripción:** Herramienta integrada en las DevTools de Chrome que permite auditar el rendimiento, accesibilidad, buenas prácticas y SEO de tu aplicación web.

**Acceso:** `Ctrl + Shift + I` > Pestaña "Lighthouse" en Chrome DevTools.

### **Webpack Bundle Analyzer**

**Descripción:** Analiza el tamaño de los paquetes generados por Webpack, ayudándote a identificar y optimizar dependencias pesadas.

```ad-note
title: Instalacion
```
```
npm install --save-dev webpack-bundle-analyzer
```

**Uso:** Configura tu proyecto Angular para generar el análisis del bundle y visualizarlo.
