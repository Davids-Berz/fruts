 "debounceTime()": permite restringir la cantidad de veces que se emiten valores de un observable en un lapso de tiempo determinado. Ejemplo:

```ad-example
title:debounceTime
```
```
import { Observable, fromEvent } from 'rxjs';
import { debounceTime } from 'rxjs/operators';

const input = document.querySelector('input');

fromEvent(input, 'input').pipe(
  debounceTime(500)
).subscribe((event) => console.log(event.target.value));
```

En este ejemplo, se utiliza el operador "debounceTime()" para restringir la cantidad de veces que se emiten valores del observable que se genera cada vez que el usuario escribe en un campo de texto.
