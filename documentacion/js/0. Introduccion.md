# Que es JavaScript?

Javascript es un lenguaje de programación interpretado y orientado a objetos utilizado principalmente para crear contenido interactivo en la web. Fue desarrollado por Netscape en 1995 y desde entonces ha evolucionado y se ha vuelto muy popular en todo el mundo. Javascript se ejecuta en el lado del cliente, es decir, en el navegador web del usuario, y permite la creación de efectos dinámicos y animaciones, la validación de formularios, la manipulación del DOM (Document Object Model), la comunicación con servidores web y muchas otras funcionalidades.

En los últimos años, el uso de Javascript se ha expandido más allá de la web, y ahora se utiliza también en el desarrollo de aplicaciones de escritorio, aplicaciones móviles, servidores y otras tecnologías. Javascript también ha dado lugar a una gran cantidad de marcos de trabajo y bibliotecas para el desarrollo web, como React, Angular y Vue, que hacen que la creación de aplicaciones web sea más rápida y sencilla. En resumen, Javascript es un lenguaje de programación poderoso y versátil que ha revolucionado la forma en que interactuamos con la web y ha ampliado nuestras posibilidades en el desarrollo de software.

# console.log

```ad-note
title:js
```
```
console.log('Hola Mundo')
```

Este código utilizará la función `console.log()` para imprimir el mensaje "Hola mundo" en la consola de Javascript. Puedes ejecutar este código en la consola del navegador web o en un entorno de desarrollo como Node.js. ¡Es una excelente manera de empezar a experimentar con Javascript!

Si quieres imprimir "Hola mundo" en una página HTML utilizando Javascript, puedes utilizar el siguiente código:

```ad-note
title:js
```
```
<!DOCTYPE html>
<html>
  <head>
    <title>Mi página web</title>
  </head>
  <body>
    <script>
      document.write("Hola mundo");
    </script>
  </body>
</html>
```
	
Este código incluye un elemento `<script>` dentro del cuerpo de la página web que ejecutará el código Javascript que escribe "Hola mundo" en la página utilizando la función `document.write()`. Cuando la página se cargue en el navegador, se imprimirá el mensaje "Hola mundo" en la página.
